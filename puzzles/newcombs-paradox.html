Newcomb's Paradox

The Scenario

You are presented with two boxes:

  • Box A: Always contains $1,000
  • Box B: Either contains $1,000,000 or nothing

A superintelligent being (the Predictor) has already predicted your choice and has either:

  • Put $1,000,000 in Box B if it predicted you would take only Box B
  • Left Box B empty if it predicted you would take both boxes

The Predictor is known to be extremely accurate in its predictions.

What do you choose?

The Paradox

This thought experiment raises questions about free will, determinism, and rational decision-making:

  • One-boxers argue: The Predictor is accurate, so taking only Box B guarantees $1,000,000
  • Two-boxers argue: The money is already in the boxes, so taking both always gives you more

Which choice do you think is more rational? Why?
